# 1. Project Abstract

The **Allostatic Friction and Hysteretic Autonomic Dynamics Framework** provides a computational and mathematical toolkit for investigating why the human nervous system frequently remains trapped within a persistent high-vigilance sympathetic attractor state despite removal of external psychological stressors.

Conventional health metrics typically monitor static organ performance.

This framework instead treats the autonomic core of the brainstem—particularly the:

* Parabrachial Nucleus (PBN)
* Nucleus Tractus Solitarius (NTS)

as a nonlinear path-dependent dynamical system.

By modeling continuous mechanical stabilization of a compromised upper airway as an **Allostatic Friction Layer**, the framework estimates energetic taxation imposed upon the nervous system's baseline metabolic budget.

A complementary **Hysteretic State Engine** demonstrates why elimination of environmental stress alone may fail to restore parasympathetic baseline activity when structural signal disturbances persist.

# 2. Theoretical Foundations

Modern physiology traditionally assumes that once a stressor disappears, biological systems naturally return to baseline.

Complex systems theory predicts otherwise.

Numerous adaptive systems demonstrate **hysteresis**, meaning their present state depends strongly upon their historical trajectory.

Examples include

* Magnetic materials
* Climate tipping points
* Financial market instability
* Neural plasticity
* Chronic pain syndromes

This repository explores whether chronic airway compensation may exhibit similar properties.

# 8. Implementation Example


```python

import numpy as np
import yaml

from core.friction_layer import AllostaticFriction
from core.hysteresis_engine import HysteresisStateEngine


def run_autonomic_simulation(duration_steps,
                             hardware_alignment_vector):


    with open("config/model_params.yaml") as f:

        params = yaml.safe_load(f)



    friction_modulator = AllostaticFriction(

        airway_volume=

        hardware_alignment_vector['airway_volume'],


        jaw_recession=

        hardware_alignment_vector['jaw_recession']

    )


    state_engine = HysteresisStateEngine(

        alpha=params['alpha'],

        beta=params['beta']

    )



    external_stress = np.zeros(duration_steps)

    midpoint = duration_steps // 2


    external_stress[:midpoint] = 0.85

    external_stress[midpoint:] = 0.0



    vigilance_history = []



    for t in range(duration_steps):


        caf = friction_modulator.compute_coefficient()


        trigeminal_flux = (

            friction_modulator

            .get_afferent_noise_flux(t)

        )


        vigilance = (

            state_engine.update_state(

                env_input=external_stress[t],

                friction_tax=caf,

                static_flux=trigeminal_flux

            )

        )


        vigilance_history.append(vigilance)



    return np.array(vigilance_history)



```
